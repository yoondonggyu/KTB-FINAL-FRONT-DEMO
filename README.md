# Devths Landing Page

Figma 디자인을 기반으로 제작된 Next.js React 랜딩 페이지입니다.

> 화면 설계서 기반 초기 프로토타입 - 빨간색 기능 정의 박스는 제외된 순수 UI

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.

### 빌드

```bash
npm run build
npm start
```

## 주요 기능

### 기능 1: 서비스(Devths) 설명 영역
- **기능 설명**: 우리 서비스에 대한 설명이 들어가는 공간
- **상세 동작**:
  - 사용자가 스와이프해서 넘길 수 있음 (터치 및 마우스 드래그 지원)
  - 동일한 슬라이드에 5초 간 머무르면 자동으로 다음 슬라이드로 이동
  - 하단 도트 네비게이션을 통해 현재 몇 번째인지 표시
  - 진행 바로 슬라이드 진행 상태 표시

### 기능 2: 시작하기 버튼
- **기능 설명**: 구글 OAuth2를 기반으로 한 로그인/회원가입 버튼
- **상세 동작**:
  - 회원이 아닌 경우 회원가입 페이지로 리다이렉트
  - 회원인 경우 로그인 후 메인 페이지(캘린더)로 이동
  - ⚠️ 현재는 핸들러만 구현되어 있으며, 실제 OAuth2 연동은 필요합니다.
